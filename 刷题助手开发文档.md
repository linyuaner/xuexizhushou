## 1. 项目概述

### 1.1 项目背景

随着在线教育的快速发展，刷题类应用已成为学生、职场人士备考和学习的重要工具。本项目旨在开发一款功能全面、用户体验优秀的"刷题助手"Web应用，结合现代Web技术栈和AI智能，提供高效的学习体验。

### 1.2 项目目标

- ✅ 开发支持响应式设计的全平台刷题助手应用
- ✅ 实现手机、平板、电脑多端无缝切换体验
- ✅ 提供多样化题目类型和智能练习模式
- ✅ 自行开发轻量级后端服务，数据自主可控
- ✅ 使用SQLite轻量级数据库，无需额外数据库服务
- ✅ 集成AI智能解析和个性化学习推荐
- ✅ 支持自定义题库管理和多格式文件导入

### 1.3 核心功能概览

### 6.1 用户认证系统

**实现要点：**
1. **表单验证**：使用 Vue 3 Composition API 实现实时验证
2. **密码安全**：bcryptjs 处理密码哈希（后端自行实现）
3. **会话管理**：JWT Token + LocalStorage 存储

**代码示例：**
```vue
<!-- LoginView.vue -->
<script setup>
import { ref } from 'vue'
import { useUserStore } from '@/stores/user'
import { useRouter } from 'vue-router'

const userStore = useUserStore()
const router = useRouter()
const email = ref('')
const password = ref('')
const loading = ref(false)
const error = ref('')

const handleLogin = async () => {
  if (!email.value || !password.value) {
    error.value = '请输入邮箱和密码'
    return
  }
  
  loading.value = true
  try {
    await userStore.login(email.value, password.value)
    router.push('/')
  } catch (err) {
    error.value = err.message || '登录失败'
  } finally {
    loading.value = false
  }
}
</script>
```

import axios from 'axios'

const api = axios.create({

  async searchQuestions(params) {

**后端实现（Express）：**
```javascript
// server/routes/questions.js
router.get('/questions', async (req, res) => {
  const { page = 1, limit = 20, search, category, difficulty, type } = req.query
  const offset = (page - 1) * limit
  
  let query = 'SELECT * FROM questions WHERE 1=1'
  const params = []
  let paramCount = 1
  
  if (search) {
    query += ` AND (title LIKE ? OR content LIKE ?)`
    params.push(`%${search}%`, `%${search}%`)
  }
  if (category) {
    query += ` AND category_id = ?`
    params.push(category)
  }
  if (difficulty) {
    query += ` AND difficulty = ?`
    params.push(difficulty)
  }
  if (type) {
    query += ` AND type = ?`
    params.push(type)
  }
  
  // 获取总数
  const countQuery = query.replace('SELECT *', 'SELECT COUNT(*) as total')
  const countResult = db.prepare(countQuery).get(...params)
  
  // 分页查询
  query += ` ORDER BY created_at DESC LIMIT ? OFFSET ?`
  params.push(limit, offset)
  const data = db.prepare(query).all(...params)
  
  res.json({
    data,
    total: countResult.total,
    page: parseInt(page),
    totalPages: Math.ceil(countResult.total / limit)
  })
})
```

### 6.3 智能练习系统

**练习模式说明：**
1. **顺序练习**：按题库顺序逐个练习
2. **随机练习**：随机抽取题目，适合复习

4. **错题练习**：针对用户错题进行强化训练
5. **易错题练习**：基于错误率统计，练习高频错题
6. **收藏练习**：练习用户收藏的题目

**练习会话管理：**
```javascript
// stores/practiceStore.js
export const usePracticeStore = defineStore('practice', {
  state: () => ({
    currentSession: null,
    sessions: [],
    currentQuestionIndex: 0,
    answers: [],
    timer: null,
    timeLeft: 0
  }),
  
  actions: {
    async startPracticeSession(bankId, practiceType, settings) {
      // 1. 创建练习会话
      const session = await practiceService.createSession({
        bank_id: bankId,
        practice_type: practiceType,
        settings
      })
      
      // 2. 加载题目
      const questions = await practiceService.getSessionQuestions(session.id)
      
      // 3. 初始化状态
      this.currentSession = session
      this.questions = questions
      this.currentQuestionIndex = 0
      this.answers = []
      
      // 4. 启动计时器（模拟考试）
      if (practiceType === 'exam' && settings.timeLimit) {
        this.startTimer(settings.timeLimit)
      }
    },
    
    async submitAnswer(questionId, answer, timeSpent) {
      // 1. 提交答案到服务器
      const result = await practiceService.submitAnswer({
        session_id: this.currentSession.id,
        question_id: questionId,
        answer,
        time_spent: timeSpent
      })
      
      // 2. 更新本地状态
      this.answers.push({
        questionId,
        answer,
        isCorrect: result.is_correct,
        timeSpent
      })
      
      // 3. 更新进度
      if (this.currentQuestionIndex < this.questions.length - 1) {
        this.currentQuestionIndex++
      } else {
        await this.completeSession()
      }
      
      return result
    }
  }
})
```

### 6.5 AI解析集成

**实现策略：**
1. **缓存优先**：优先使用缓存的解析结果
2. **异步生成**：AI解析在后台进行，不影响用户操作
3. **质量反馈**：用户可对解析质量进行评分

**AI服务封装：**
```javascript
// services/aiService.js
export const aiService = {
  cache: new Map(),
  
  async getExplanation(questionId, questionData) {
    // 1. 检查缓存
    const cached = this.cache.get(questionId)
    if (cached && Date.now() - cached.timestamp < 7 * 24 * 60 * 60 * 1000) {
      return cached.explanation
    }
    
    // 2. 调用AI API
    const explanation = await this.generateExplanation(questionData)
    
    // 3. 缓存结果
    this.cache.set(questionId, {
      explanation,
      timestamp: Date.now()
    })
    
    // 4. 保存到数据库
    await this.saveToDatabase(questionId, explanation)
    
    return explanation
  },
  
  async generateExplanation(questionData) {
    const prompt = `
      请解析以下题目：
      
      题目：${questionData.title}
      内容：${questionData.content}
      类型：${questionData.type}
      难度：${questionData.difficulty}
      正确答案：${JSON.stringify(questionData.answer)}
      
      请提供：
      1. 解题思路
      2. 详细步骤
      3. 相关知识点
      4. 易错点提醒
      5. 类似题目建议
    `
    
    const response = await openai.chat.completions.create({
      model: 'gpt-3.5-turbo',
      messages: [
        { role: 'system', content: '你是一个专业的题目解析助手。' },
        { role: 'user', content: prompt }
      ],
      temperature: 0.7,
      max_tokens: 1000
    })
    
    return response.choices[0].message.content
  }
}
```

### 15.3 API 开发指南

**自定义后端服务封装：**
```javascript
// api/index.js
import axios from 'axios'

const api = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || 'http://localhost:3000/api',
  timeout: 10000
})

// 请求拦截器 - 添加Token
api.interceptors.request.use(
  (config) => {
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  (error) => Promise.reject(error)
)

// 响应拦截器 - 处理Token过期
api.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      localStorage.removeItem('token')
      window.location.href = '/login'
    }
    return Promise.reject(error)
  }
)

export default api
```

**后端服务示例（Express）：**
```javascript
// server/index.js
const express = require('express')
const cors = require('cors')
const Database = require('better-sqlite3')
const jwt = require('jsonwebtoken')
const bcrypt = require('bcryptjs')

const app = express()
const db = new Database('quiz.db')
const SECRET_KEY = process.env.JWT_SECRET || 'your-secret-key'

app.use(cors())
app.use(express.json())

// 数据库初始化
db.exec(`
  CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    email TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    username TEXT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
  )
`)

// 认证路由
app.post('/api/auth/register', async (req, res) => {
  const { email, password, username } = req.body
  
  const passwordHash = await bcrypt.hash(password, 10)
  
  try {
    const stmt = db.prepare(
      'INSERT INTO users (email, password_hash, username) VALUES (?, ?, ?)'
    )
    const result = stmt.run(email, passwordHash, username)
    
    const token = jwt.sign({ userId: result.lastInsertRowid }, SECRET_KEY, { expiresIn: '7d' })
    
    res.json({ success: true, token, userId: result.lastInsertRowid })
  } catch (error) {
    res.status(400).json({ success: false, message: '注册失败' })
  }
})

app.listen(3000, () => {
  console.log('服务器运行在 http://localhost:3000')
})
```

### 15.4 状态管理（Pinia）

**Store 定义示例：**
```javascript
// stores/user.js
import { defineStore } from 'pinia'
import { authService } from '@/services/supabase'

export const useUserStore = defineStore('user', {
  state: () => ({
    user: null,
    isLoggedIn: false,
    loading: false
  }),
  
  actions: {
    async login(email, password) {
      this.loading = true
      try {
        const { data, error } = await authService.signIn(email, password)
        if (error) throw error
        this.user = data.user
        this.isLoggedIn = true
      } catch (error) {
        throw error
      } finally {
        this.loading = false
      }
    }
  }
})
```
