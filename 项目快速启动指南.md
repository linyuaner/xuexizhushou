# 刷题助手项目快速启动指南

## 🚀 快速开始

### 1. 环境准备

```bash
# 检查 Node.js 版本
node --version  # 需要 18+ 或 20+

# 检查 npm 版本
npm --version   # 需要 9+

# 如果没有 Node.js，请安装：
# 访问 https://nodejs.org 下载安装
```

### 2. 项目启动

```bash
# 1. 克隆项目（如果尚未克隆）
git clone <项目地址>
cd 学习助手

# 2. 安装前端依赖
npm install

# 3. 安装后端依赖
cd server
npm install
cd ..

# 4. 配置环境变量
# 前端 .env 文件
# VITE_API_BASE_URL=http://localhost:3000/api

# 后端 server/.env 文件
# PORT=3000
# JWT_SECRET=your-super-secret-key-at-least-32-characters
# DATABASE_PATH=./data/quiz.db

# 5. 启动后端服务
cd server
npm start

# 6. 新开终端，启动前端服务
npm run dev

# 7. 在浏览器中打开
# http://localhost:5173
```

### 3. 数据库配置

项目使用 sql.js（纯JavaScript实现的SQLite），无需额外安装数据库：
- 数据库文件自动创建在 `server/data/quiz.db`
- 首次启动会自动创建表结构和示例数据
- 默认管理员账户：用户名 `admin`，密码 `admin123`

### 4. 常用命令

```bash
# 前端开发命令
npm run dev          # 启动前端开发服务器
npm run build        # 构建生产版本
npm run preview      # 预览生产构建

# 后端开发命令
cd server
npm start            # 启动后端服务器
npm run dev          # 启动后端服务器（带热重载）
```

## 🎯 核心功能使用

### 用户认证
- 访问 `/login` 登录
- 默认管理员：用户名 `admin`，密码 `admin123`
- 登录后右上角显示用户菜单

### 题目浏览
- 访问 `/questions` 浏览题目
- 支持分类筛选、难度筛选
- 支持关键词搜索

### 开始练习
1. 访问 `/practice` 选择题库
2. 选择练习模式：
   - 顺序练习：按顺序做题
   - 随机练习：随机抽题
   - 模拟考试：限时答题
   - 错题练习：强化错题
   - 易错题练习：练习高频错题
   - 收藏练习：练习收藏题目
3. 开始答题，系统自动记录进度

### 题库管理
- 访问 `/banks` 管理题库
- 创建自定义题库
- 支持Excel文件导入题目

### 个人中心
- 访问 `/profile` 查看学习统计
- 查看收藏题目、学习笔记
- 管理个人资料

## 🔧 开发工作流

### 1. 创建新功能
```bash
# 创建新页面
touch src/views/NewFeatureView.vue

# 创建新组件
touch src/components/common/NewComponent.vue

# 创建新API服务（在现有api/index.js中添加）
```

### 2. 添加路由
```javascript
// src/router/index.js
{
  path: '/new-feature',
  name: 'NewFeature',
  component: () => import('@/views/NewFeatureView.vue'),
  meta: { requiresAuth: true }  // 需要登录
}
```

### 3. 状态管理
```javascript
// src/stores/newStore.js
import { defineStore } from 'pinia'

export const useNewStore = defineStore('new', {
  state: () => ({
    data: null,
    loading: false
  }),
  
  actions: {
    async fetchData() {
      this.loading = true
      // API 调用
      this.loading = false
    }
  }
})
```

## 🐛 常见问题解决

### 1. 启动失败
```bash
# 清除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install

# 后端同样操作
cd server
rm -rf node_modules package-lock.json
npm install
```

### 2. 数据库问题
```bash
# 数据库文件位置
# server/data/quiz.db

# 重置数据库（删除后重启服务会自动重建）
rm server/data/quiz.db
cd server
npm start
```

### 3. 端口冲突
```bash
# 修改后端端口（server/.env）
PORT=3001

# 修改前端API地址（.env）
VITE_API_BASE_URL=http://localhost:3001/api
```

### 4. 样式问题
```bash
# 重新构建
npm run build
```

## 📱 响应式设计

### 断点说明
